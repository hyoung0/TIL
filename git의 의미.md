## git terminal
### add, commit의 의미
- (1통) untracked files : 트래킹되지 않은 파일들
    
    1.txt를 만들었지만, add를 하지 않음
- (2통) changes to be committed : 커밋될 변경사항들

    보고서.txt 만들고 add함
